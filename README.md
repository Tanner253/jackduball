# Jack DuBall

Roll down the lane as Jack Duval and smash trees and houses — avoid the spikes!

## Play locally

Open `index.html` in a browser, or serve the folder with any static server.

## Deploy

```bash
vercel --prod
```

## Controls

- **A / D** or **← / →** — steer
- **Tap** left or right side of screen on mobile
